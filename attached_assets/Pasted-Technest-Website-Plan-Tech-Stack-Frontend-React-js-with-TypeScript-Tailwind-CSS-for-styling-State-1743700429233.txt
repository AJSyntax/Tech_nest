Technest Website Plan
Tech Stack

Frontend: React.js with TypeScript, Tailwind CSS for styling
State Management: Context API or Redux for smaller scale
Form Handling: React Hook Form with Yup for validation
File Handling: File-saver.js for ZIP downloads
UI Components: Headless UI or Shadcn/UI for accessible components
Responsive Design: Tailwind's responsive utilities
Image Processing: Sharp or browser-native APIs
Color Picker: React Color
Preview Engine: Iframe-based live preview
